# RayMack 亞維機械｜臨時官網專案說明

本專案為 **亞維機械股份有限公司（RayMack）**
之「臨時官方網站」，用於正式官網完成前的過渡使用。

主要目的為配合展覽、產品展示與客戶初步資訊查詢。

---

## 🔧 使用技術

- HTML5
- Bootstrap 5
- CSS / Sass
- JavaScript（少量）
- GitHub Pages（部署）

---

## 🌐 官網用途

本臨時官網包含以下功能：

- 公司簡介
- 平面磨床產品清單
- 產品詳細介紹頁（如 SGS-1020AHD）
- 公司公告（新聞／展覽／新品）
- 聯絡資訊與洽詢管道
- 基本 SEO 架構

---

## 📁 頁面結構

- `index.html`：首頁
- `about.html`：關於我們
- `product-list.html`：產品清單
- `product-detail.html`：產品介紹頁
- `news-list.html`：公司焦點（公告／展覽）
- `contact.html`：聯絡我們

---

## 🚀 部署方式

- 使用 **GitHub Pages** 架設
- 未來將指向公司購買之正式網域
